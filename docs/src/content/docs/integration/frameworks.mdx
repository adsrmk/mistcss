---
title: Frameworks
---

By default, `mistcss` will generate React components that you can use in Next.js or Remix

## React

```sh
mistcss ./components
```

## Vue

```sh
mistcss ./components --target=vue
```

## Astro

```sh
mistcss ./components --target=astro
```

## Hono

```sh
mistcss ./components --target=hono
```
